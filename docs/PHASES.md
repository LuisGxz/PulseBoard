## Stack
- **Front:** Angular 20 (standalone + signals) + Tailwind v4 + ApexCharts. Grid de widgets arrastrables (drag & drop).
- **Back (orquestación):** .NET 9 Web API (Clean Architecture) — auth, dashboards/widgets CRUD, datasets, orquesta el ETL y sirve queries agregadas.
- **Microservicio ETL:** Python FastAPI + pandas — ingesta CSV, procesa, agrega y escribe en Postgres.
- **DB:** PostgreSQL 18 (datos analíticos + metadatos).
- **Auth:** JWT + refresh + RBAC (Owner / Editor / Viewer por dashboard).
- **Deploy:** Docker Compose multi-servicio (dev). Deploy en nube diferido.

## Cómo correr (dev)
1. `docker compose up -d` (Postgres 18 en `localhost:55432`).
2. `cd backend && dotnet run --project src/PulseBoard.Api` → migra + seed automático. API en `http://localhost:5180` (`/health`, `/api/datasets`, `/api/auth/*`).
3. ETL Python: `docker compose up -d etl` (o local: `cd etl && python -m venv .venv && .venv/Scripts/pip install -r requirements-dev.txt && .venv/Scripts/python -m uvicorn app.main:app --port 8100`). Tests: `.venv/Scripts/python -m pytest`. Health en `http://localhost:8100/health`.
4. Front: `cd frontend && npx ng serve` → `http://localhost:4200` (API base por defecto `localhost:5180`). Build: `npx ng build`. Screenshots de verificación: `node scripts/shot.mjs` (Playwright).
